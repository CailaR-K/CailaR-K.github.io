Space Pirates

Creating a dungeon-crawler game to allow players to succeed.

We designed and created a single player dungeon crawler game featuring AI agents for path planning. The game was generated through infinite procedural content, with adjustable difficulty to help players succeed. The development spanned one semester, involving a six-member team working with the C# programming language. My contributions included the game's overall design concept, visual arts production, audio elements, testing, as well as the coding of five distinct reward categories (Access, Facility, Sustenance, Glory, Praise).

Game Description

This game is a third-person dungeon crawler in an abandoned space outpost. As a lonely astronaut you must escape the mysterious space station before the lights go out and the enemy finds you.
